# EDK — 용어 사전 (Terminology)

> 원본: pmg-ic-pbl/docs/terminology.md
> 상태: EDK 전환 중 — 업데이트 필요

---

## 1. 제품 도메인

| 한국어 | 변수명 | 클래스명 | 정의 |
|--------|--------|---------|------|
| 의약품/영양제 | `medicine` | `Medicine` | 판매 가능한 일반의약품 또는 영양제 1종 |
| 증상 카테고리 | `symptom_category` | `SymptomGroup` | 두통/감기/소화불량/피로/외상 등 분류 |
| 증상 항목 | `symptom` | `Symptom` | 개별 증상 선택 항목 |
| 효능 | `description` | — | 의약품의 주요 효과 설명 |
| 복용법 | `dosage` | — | 복용 방법 및 용량 안내 |
| 주의사항 | `caution` | — | 복용 시 주의해야 할 사항 |

---

## 2. 취급 품목 기준

| 구분 | 취급 | 예시 |
|------|------|------|
| 영양제 / 건강기능식품 | ✅ | 비타민, 오메가3, 박카스 |
| 일반의약품 | ✅ | 타이레놀, 판콜에이, 마데카솔 |
| 전문의약품 | ❌ | 항생제, 수면제 (처방전 필요) |

---

## 3. 재고 도메인 (재활용)

| 한국어 | 변수명 | 정의 |
|--------|--------|------|
| 재고 | `stock` | 의약품/영양제 현재 보유 수량 |
| 최대 용량 | `max_capacity` | 최대 보유 가능 수량 |
| 품절 | `out_of_stock` | 재고 0 — UI에서 선택 불가 처리 |

---

## 4. 장바구니 / 결제 도메인 (재활용)

| 한국어 | 변수명 | 정의 |
|--------|--------|------|
| 장바구니 | `cart` | 구매 전 담은 제품 목록 |
| 주문 항목 | `order_item` | 제품 1종 + 수량 + 소계 |
| 결제 금액 | `final_amount` | 최종 청구 금액 |
| 잔돈 | `change_amount` | 투입 금액 - 결제 금액 |
| 잔돈 보유량 | `change_reserve` | 키오스크 내 권종별 보유 수량 |

---

## 5. 관리자 도메인 (재활용)

| 한국어 | 변수명 | 정의 |
|--------|--------|------|
| 관리자 | `admin` | 제품 정보/재고/설정 접근 권한자 |
| 관리자 비밀번호 | `admin_password` | JSON에 저장 |
| 매출 | `revenue` | 결제 완료된 금액의 누적 합산 |

---

## 6. 예외 도메인 (재활용)

| 예외 | 발생 조건 |
|------|---------|
| `StockOverflowException` | 재고 보충 시 max_capacity 초과 |
| `InsufficientChangeException` | 잔돈 반환 불가 |
| `PaymentException` | 결제 처리 실패 |
| `InsufficientStockException` | 재고 차감 시 부족 |
| `AdminAuthException` | 관리자 비밀번호 불일치 |
| `InvalidRecipeException` | 데이터 JSON 형식 오류 |

---

## 7. 데이터 파일 구성

| 파일 | 내용 | 갱신 시점 |
|------|------|---------|
| `medicines.json` | 제품 목록, 가격, 재고, 증상 카테고리 | 구매 완료 / 관리자 수정 시 |
| `symptoms.json` | 증상 카테고리 정의 | 관리자 수정 시 |
| `change_reserve.json` | 권종별 현금 보유량 | 결제 완료 / 관리자 수정 시 |
| `admin_config.json` | 관리자 비밀번호 | 관리자 수정 시 |
