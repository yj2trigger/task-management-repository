## 🧠 핵심 비즈니스 로직: 3-Mode State Machine

> 기준: 해당 성별의 전체 이용 가능 세탁기 수

| 모드 | 조건 | 동작 |
|------|------|------|
| **Mode A** | 4대 이상 | 층별 이용 가능 세탁기 수 표시, 사용자가 직접 판단 |
| **Mode B** | 1~3대 | 버튼 누르면 1:1로 세탁기 위치 안내 + 10분 소프트 예약 |
| **Mode C** | 0대 | 대기열 등록 → 빈 자리 발생 시 순서대로 알림, 10분 미사용 시 다음 대기자에게 전달 |
